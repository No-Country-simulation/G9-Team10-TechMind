# Fase 3: Clasificación e Integración de LLM

Esta carpeta contiene los modelos de Machine Learning encargados de dotar de inteligencia al proyecto.

## Tareas principales de esta fase:
1. **Modelo de Clasificación (ML Clásico):** Entrenar un algoritmo rápido (ej. Scikit-Learn, Random Forest, Naive Bayes) capaz de leer un texto limpio y categorizarlo (ej. Frontend, Backend, IA, DevOps).
2. **Extracción de Entidades (LLM):** Scripts de *Prompt Engineering* que se conectan a APIs de Inteligencia Artificial (Gemini/OpenAI) para extraer dinámicamente "palabras clave" (*tags*) y generar un resumen conciso del texto.
