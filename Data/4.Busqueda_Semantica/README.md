# Fase 4: Búsqueda Semántica y Recomendación

Esta carpeta aloja los algoritmos de recomendación matemática, cumpliendo con el requisito de "Encontrar contenidos relacionados".

## Tareas principales de esta fase:
1. **Vectorización:** Conversión del catálogo de textos a vectores matemáticos (usando TF-IDF o Embeddings densos).
2. **Similitud del Coseno:** Implementación de la métrica de distancia para comparar un documento nuevo contra todo el catálogo existente.
3. **Motor de Recomendación:** Lógica para ordenar los resultados y devolver los "Top 3" IDs más parecidos al texto analizado.
