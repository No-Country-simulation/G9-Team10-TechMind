# Fase 2: Limpieza de Datos (Data Wrangling)

Esta fase se encarga de preparar el dataset crudo proveniente de la Fase 1 para su posterior análisis.

## Objetivo

Aplicar procesos de limpieza, normalización y estandarización sobre los datos para generar una versión más consistente y utilizable del dataset.

## Tareas realizadas

- Limpieza de etiquetas HTML como `<p>`, `<b>`, entre otras.
- Enmascaramiento de bloques de código fuente con tokens como `<CODE_BLOCK>`.
- Estandarización de mayúsculas y minúsculas.
- Eliminación de caracteres corruptos o inconsistentes.
- Generación del archivo `dataset_techmind_clean.csv`.

## Contenido de esta carpeta

- `Datos_limpios.ipynb`: notebook con el proceso de limpieza.
- `dataset_techmind_original.csv`: dataset original sin procesar.
- `dataset_techmind_clean.csv`: dataset limpio resultante.

## Resultado

El resultado de esta fase es un dataset depurado y normalizado, listo para continuar con el análisis exploratorio o la siguiente etapa del proyecto.
