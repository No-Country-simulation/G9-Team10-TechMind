# Fase 2: Limpieza de Datos (Data Wrangling)

Esta carpeta está destinada a contener los notebooks (ej. Google Colab) y scripts encargados de tomar el dataset crudo de la Fase 1 y normalizarlo.

## Tareas principales de esta fase:
- Limpieza de etiquetas HTML (`<p>`, `<b>`, etc.).
- Enmascaramiento de bloques de código fuente (reemplazo por tokens como `<CODE_BLOCK>`).
- Estandarización de mayúsculas/minúsculas y eliminación de caracteres corruptos.
- Generación del archivo `dataset_techmind_clean.csv`.
