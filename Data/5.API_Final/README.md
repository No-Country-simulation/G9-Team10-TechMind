# Fase 5: API Final (El Motor de IA)

Esta carpeta contendrá el producto final de nuestro equipo de Datos: una API encapsulada lista para ser consumida por el Backend principal (Spring Boot).

## Tareas principales de esta fase:
- **Endpoints de FastAPI / Flask:** 
  - `/analizar_texto`: Recibe texto crudo y devuelve Categoría, Probabilidad y Tags en JSON (conecta con la Fase 3).
  - `/buscar_parecido`: Recibe un texto y devuelve los IDs de los documentos recomendados (conecta con la Fase 4).
- **Integración:** Unir los scripts sueltos de las fases anteriores en un pipeline orientado a objetos, robusto y preparado para producción.
